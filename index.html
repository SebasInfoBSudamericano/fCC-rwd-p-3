Introduction

This document is designed for individuals who want to learn Python from scratch. We will cover the basics through five progressive chapters.

By the end, readers will have a solid foundation to write simple programs and understand the core features of the language.

Chapter 1

Installation and Environment Setup

Objective:

Set up a suitable development environment to work with Python.

Contents:
  1. Download and Install Python:
    • Visit python.org and download the latest version.
    • Ensure to include Python in the PATH during installation.
  2. Install an IDE:
    • Recommendations:
      • Visual Studio Code (VS Code).
      • PyCharm Community Edition.
      • Thonny (ideal for beginners).
  3. Basic Setup:
    • Create and execute the first .py file.
    • Use pip to install additional packages.
  4. Test Installation:
    • Open the terminal and run:
      bash 
      
python --version
    • Create a hello_world.py file with the following content:
      python 
      
print("Hello, World!")
Chapter 2

Basic Syntax and Control Structures

Objective:
Understand Python's basic syntax and how to control program flow.
Contents:
  1. Basic Syntax:
    • Indentation and comments.
    • Variables and data types:
      python 
      
name = "John"
age = 25
  2. Control Structures:
    • Conditionals:
      python 
      
if age >= 18: 
	print("You are an adult.")
else:
	print("You are a minor.")
    • Loops:
      • for
        python 
        
for i in range(5):
	print(i)
      • while
        python 
        
counter = 0
while counter < 5:
	print(counter)
	counter += 1
  3. Input and Output:
    python 
    
name = input("What's your name?")
print(f"Hello, {name}!")
Chapter 3

Functions and Error Handling

Objective:
Learn to modularize code using functions and handle common errors.
Contents:
  1. Defining and Using Functions:
    python 
    
def greet(name):
	print(f"Hello, {name}!")

greet("Anna")
  2. Parameters and Return Values:
    python 
    
def add(a, b):
	return a + b

result = add(3, 5)
print(result)
  3. Error Handling with try-except
    python 
    
try:
	number = int(input("Enter a number: "))
	print(f"The double is {number * 2}")
except valueError:
	print("You must enter a valid number")
Chapter 4

Introduction to Collections

Objective:
Master the use of lists, tuples, sets, and dictionaries.
Contents:
  1. Lists:
    • Creating and manipulating lists:
      python 
      
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits)
  2. Tuples:
    • Immutable collections:
      python 
      
colors = ("red", "green", "blue")
print(colors[0])
  3. Sets:
    • Set operations:
      python 
      
numbers = {1, 2, 3, 4}
numbers.add(5)
print(numbers)
  4. Dictionaries:
    • Key-value pairs:
      python 
      
person = {"name": "Carlos", "age": 30}
print(person["name"])
